**EPC Project Cost Control & Budget Variance Analysis**

**Objective:**
- To compare budget vs actual cost and identify cost overruns in EPC projects.

**Tools Used:**

- Excel (Data cleaning, Pivot Tables)

- Power BI (Dashboards, KPIs)

**Work Done:**

- Cleaned and validated cost data in Excel

- Created pivot tables for project-wise and monthly summaries

- Calculated cost variance and variance percentage

- Built Power BI dashboards for management review

**Key Insights:**

- Identified over-budget projects

- Highlighted high-risk projects using variance %

- Analyzed monthly cost trends

**Business Value:**
- Helped management identify cost overruns early and improve cost control.
