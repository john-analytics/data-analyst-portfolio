# EPC Project Progress Tracking & Delay Risk Analysis

**Objective:**
- To track planned vs actual project progress and identify delays.

**Tools Used:**

- Excel (Data validation, Pivot Tables)

- Power BI (Progress & Risk Dashboards)

**Work Done:**

- Prepared and cleaned progress data in Excel

- Calculated progress gap and delay days

- Classified projects into Low, Medium, and High risk

- Built dashboards to track execution status

**Key Insights:**

- Identified delayed and high-risk projects

- Analyzed delay trends by month and location

**Business Value:**
- Supported early decision-making to reduce project delays.
