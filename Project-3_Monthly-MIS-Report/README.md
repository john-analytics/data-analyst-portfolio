## Monthly MIS Reporting & Performance Analysis for EPC Projects

### Objective
To prepare monthly MIS reports for management review.

### Tools Used
- Excel (MIS preparation, Pivot Tables)
- Power BI (Management Dashboards)

### Work Done
- Cleaned and structured MIS data in Excel
- Created monthly, project-wise, and department-wise summaries
- Built Power BI dashboards to track expenses, revenue, profit, and manpower

### Key Insights
- Compared department performance
- Identified profitable projects

### Business Value
Helped management review monthly performance and support data-driven decisions.
